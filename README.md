# D&D 2024 Character Sheet

A fully featured D&D 2024 PHB character sheet app. Built with React + Vite.

**Live site:** https://YOUR_GITHUB_USERNAME.github.io/dnd-sheet

## Features
- Full character creation (class, species, background, abilities, skills, feats)
- Auto-calculated HP, AC, attack bonuses, spell slots
- Equipment tracker with armour/weapon management
- Spell management with per-class limits
- PDF export (drawn from scratch, no template needed)
- Multiple character roster saved to localStorage

## Local development

```bash
npm install
npm run dev
```

## Deploy to GitHub Pages

```bash
npm run deploy
```

This builds the project and pushes the `dist/` folder to the `gh-pages` branch automatically.
